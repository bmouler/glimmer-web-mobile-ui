About Glimmer (from release notes):

Glimmer is a collection of programs for identifying genes in microbial DNA sequences. The system works by creating a variable-length Markov model from a training set of genes and then using that model to attempt to identify all genes in a given DNA sequence. Version 1 of Glimmer was described in [SDKW98] and Version 2 was described in [DHK+99].

Glimmer3 is released as OSI Certified Open Source Software under the Artistic License.
